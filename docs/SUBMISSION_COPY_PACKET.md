# Submission Copy Packet

## Short Description

Goose Bounty/Grant Scout MCP is a Goose extension that helps developers rank bounty and grant opportunities by payout collectability, Goose-assisted autonomy, and delivery risk, then prepare grant or bounty submission materials.

## Public Repo

https://github.com/sevencat2004/goose-bountypay-scout

## Requested Grant

$48,000 USD over 4 months.

## Core Deliverables

- Goose-compatible STDIO MCP extension.
- Opportunity scoring by collectability, autonomy, and risk.
- Public GitHub issue analyzer.
- Source adapters for bounty and grant programs.
- Markdown/JSON scout reports.
- Submission package and payout-risk tracker.
- Goose setup docs and demo prompts.

## Why It Matters

Open-source contributors waste time chasing rewards that are already assigned, duplicated, stale, blocked by proposal gates, or incompatible with AI-assisted workflows. This project gives Goose users a practical way to identify fundable work before investing engineering time.
