# Roadmap

## Current MVP

- STDIO MCP server for Goose.
- Deterministic opportunity scoring.
- Public GitHub issue live analyzer.
- Public GitHub issue search.
- Markdown scout report generation.
- Grant application draft generator.
- Self-test, MCP smoke test, and preflight.

## Grant Milestone Plan

### M1 - Working MCP Extension MVP

Status: mostly complete locally.

Remaining polish:

- Publish public GitHub repo.
- Add screenshots or terminal demo transcript.
- Add install troubleshooting notes for Windows and macOS/Linux.

### M2 - Live Source Adapters And Report Workflow

Planned:

- GitHub issue search adapter.
- Algora-style board parser or GitHub label/payout inference.
- Manual grant-program source adapter.
- JSON report writer and persisted report files.
- Source health and failure reporting.

### M3 - Submission Assistant And Payment-Risk Tracker

Planned:

- Submission package generator for grants and bounties.
- Follow-up queue for submitted work.
- Payout/account action checklist per project.
- Central project-record export/import helpers.

### M4 - Community Handoff

Planned:

- Goose install guide.
- Example Goose sessions.
- Issue templates.
- Maintenance docs.

## Non-Goals

- The tool will not auto-submit grant applications, bounty claims, GitHub comments, private vulnerability reports, KYC forms, tax forms, or wallet/payment details.
- The tool will not assist with programs that prohibit AI-assisted research or submissions.
